# 📊 PR. Final Project – Data Intelligence Dashboard

## 📌 Project Overview
The **Data Intelligence Dashboard** is an advanced Microsoft Excel analytics project built to analyze customer transaction data and generate meaningful business insights through automation, formulas, PivotTables, charts, and interactive dashboards.

This project demonstrates practical implementation of Excel features used in real-world business intelligence and data analysis workflows.

---

# 🎯 Objective
To create a complete Excel-based reporting and visualization solution that:

- Cleans and processes raw transaction data
- Performs automated calculations
- Generates customer and sales insights
- Builds dynamic reports and dashboards
- Uses advanced Excel functions and tools

---

# 📂 Dataset Information
The project uses the sheet:

### `Final Project Dataset`

Containing:
- 200+ customer transaction records

- Sales data
